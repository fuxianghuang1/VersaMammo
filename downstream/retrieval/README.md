## Usage

1. Extract features by running `extract_features.py`
2. Merge multiple datasets by running `feature4retrieval.py`
3. Test the retrieval task by running `test_retrieval_final.py`

If you want to directly test the retrieval task, please download the pre-extracted features and extract them to the current folder.
- [Download Features](https://drive.google.com/file/d/1y5PAYLnqPjEq-wnSGq55C3V-gNSa2nFi/view?usp=drive_link)
  
Please adjust your data paths to ensure proper execution.
